# まいにち牧場 要件定義

## 1. 概要

小学生の毎日の支度・帰宅後のやること・寝る準備を、牧場ゲームと組み合わせて習慣化するアプリ。

子どもはミッションを達成してコインを獲得し、動物、えさ、水、特別なお世話アイテムを購入する。動物は毎日お世話しないと弱っていき、丁寧に世話を続けると恩返しとしてクーポンを残して旅立つ。

## 2. 目的

- 朝・昼・夜のルーティンを自分で進められるようにする
- 毎日のタスク継続に意味を持たせる
- コイン、動物、世話、恩返しによって継続意欲を高める
- 親の声かけ負担を減らす
- 家庭内クーポンを自然なごほうびとして扱う

## 3. 主要機能

### ミッション

- 朝、昼、夜のカテゴリでタスクを表示する
- チェックを付ける操作は保護者認証なしでできる
- チェックを取り消す操作は保護者コード承認を必要にする
- ミッションごとにコイン報酬を設定できる
- 1日の全ミッション達成でボーナスコインを付与する

### コイン

- コインはアプリ内通貨として扱う
- どうぶつガチャ、えさ、水、特別えさ、薬、ブラシ、おもちゃ、ベッドの利用に使う
- 実際のお金や課金とは結びつけない

### 動物

- うし、にわとり、ひつじ、うさぎ、やぎ、いぬ、ぶた、あひる、うま、ねこ、アルパカを候補とする
- 動物はどうぶつガチャで出会うと牧場の柵の中に現れる
- 動物にはレアリティを持たせ、レアリティが高いほど出会いにくくする
- どうぶつガチャには3〜5秒程度の明るい演出を持たせ、スキップできる
- ガチャ演出はステートを分けて管理する
- ガチャ演出中はBGM音量を少し下げ、終了後に戻す
- ガチャ結果では `NEW!`、子ども向けレアリティ表示、結果アクションを表示する
- 複数の動物を保有している場合、左右ボタンで表示を切り替える
- 表示中の動物が何匹中何匹目かを、牧場背景の左上に表示する
- 動物には名前を付けられ、名前は牧場画像内ではなく詳細エリアに表示する
- 動物の名前変更には保護者コード承認を必要にする
- 動物にはお腹、水分、体調、なつきの状態を持たせる
- 動物画像は状態に応じて normal、happy、hungry、sick、wary、farewell を切り替える

### 世話

- えさをあげる
- 水をあげる
- なでる
- 特別えさでお腹、体調、なつきを大きく回復できる
- 薬で体調を回復できる
- ブラシやおもちゃは、なでる代わりのお世話として扱う
- ベッドで休ませて体調を回復できる
- 1日に3つの世話を完了するとお世話記録が1日増える
- 世話をしない日が続くと、お腹、水分、体調、なつきが下がる
- 状態が悪いと体調不良、警戒、逃亡が起きる

### 恩返しクーポン

- お世話記録が十分にたまり、なつきと体調が高い動物は恩返しクーポンを残す
- クーポンを残した動物は牧場から旅立つ
- クーポンは未使用・使用済みを管理できる
- クーポン使用状態の変更には保護者コード承認を必要にする
- 保護者はクーポン候補の名前と内容を編集できる

### アルバム

- 恩返しや逃亡など、牧場からいなくなった動物の記録を残す
- 記録には動物の見た目、名前、日付、理由を表示する
- 実スクリーンショットではなく、アプリ内の記録カードとして保存する

### 保護者設定

- 保護者コードを設定できる
- ミッションの追加・編集・削除ができる
- クーポン候補の追加・編集・削除ができる
- 重要操作には保護者コード承認を必要にする
- 設定タブは通常のタブには表示せず、右上の三本線メニューを開く時点で保護者コード承認を必要にする

### ネタバレ防止

- プレゼント画面は、クーポン獲得前には恩返しやお礼を直接示す文言を表示しない
- アルバムタブは、記録ができるまで表示しない

### サウンド

- `assets/audio/bgm/bgm-main.mp3` をBGMとしてループ再生する
- BGMの初期音量は30%とする
- BGMのON/OFF設定を持つ
- SEのON/OFF設定を持つ
- BGM音量とSE音量を別々に管理する
- ボタン押下時に `se-tap.mp3` を再生する
- 画面やタブの切替時に `se-transition-02.mp3` を再生する
- ガチャ演出の開始、開封、レアリティ結果に対応する効果音を再生する
- ルーチン完了時に `se-complete.mp3` を再生する
- スタンプ獲得時に `se-stamp.mp3` を再生する
- ごほうび表示時に `se-reward.mp3` を再生する
- 全タスク完了時に `se-all-clear.mp3` を再生する
- 音声設定は `localStorage` に保存する
- ページ読み込み直後に音ありBGMを再生しない
- トップ画面の「はじめる」ボタン押下を最初のユーザー操作として扱い、BGMを開始してアプリ本体へ進む
- BGM再生や音源読み込みに失敗した場合は、原因をブラウザのコンソールに出力する

### PWA

- GitHub PagesやlocalhostなどHTTP(S)配信時にホーム画面へ追加できる
- アプリ本体、主要画像、表示済み画像をキャッシュしてオフライン起動に対応する
- `file://` で直接開く場合は通常の静的アプリとして動作し、Service Worker登録は行わない

## 4. MVP範囲

- 静的HTML/CSS/JavaScriptで動作
- PWA対応
- localStorage保存
- ミッションチェック
- コイン獲得
- どうぶつガチャ
- 動物レアリティ
- えさ・水・お世話アイテム購入
- 動物の世話
- 日次の状態低下
- 恩返しクーポン
- アルバム
- 保護者コード
- 保護者メニュー
- BGM / SE
- 画像素材による牧場、動物、アイテム、カレンダー、クーポン表示

## 5. 今後の拡張案

- 牧場の柵、背景、装飾を購入できる
- 動物小屋、畑、池、風車を追加できる
- えさを畑で育てられる
- 動物ごとの性格、好きな食べ物、イベントを追加する
- 季節や天気によるイベントを追加する
- 季節イベントや限定動物を追加する
- 親向け週次レポートを追加する

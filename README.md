# まいにち牧場

小学生向けの生活習慣チェックと牧場育成を組み合わせたブラウザアプリです。

毎日のミッションを達成するとコインを獲得できます。コインで動物、えさ、水を購入し、動物を毎日お世話します。

## できること

- 朝、昼、夜のミッションをチェック
- ミッション達成でコインを獲得
- コインでどうぶつガチャをまわせる
- どうぶつガチャでは短い演出のあとに動物が登場する
- コインでえさ・水・特別えさ・薬・ブラシ・おもちゃ・ベッドをショップで用意できる
- 動物にえさ、水、なでる、ブラシ、遊び、休憩などの世話をする
- 11種類の動物を迎えられる
- 動物にレアリティがある
- 動物に名前を付け直せる
- 動物の状態に合わせて画像が変わる
- 季節や時間帯に合わせて牧場背景が変わる
- 世話をしない日が続くと、お腹・水分・体調・なつきが下がる
- 体調が悪くなったり、警戒したり、逃げてしまうことがある
- しっかり世話を続けると動物が恩返しクーポンをくれる
- 恩返しやお別れはアルバムに記録される
- クーポン内容とミッション内容は保護者が編集可能
- チェックを付ける操作は認証不要、チェック取り消しや右上の保護者メニューは保護者コードが必要
- GitHub Pagesやlocalhost配信時にPWAとしてホーム画面へ追加できる
- BGMと効果音を再生でき、音量とON/OFFを保存できる

## 使い方

`index.html` をブラウザで開くと使えます。サーバーやインストールは不要です。

PWAとしてインストールする場合は、GitHub Pagesやlocalhostなど `http://` または `https://` で配信してください。`file://` で直接開いた場合、ブラウザ仕様によりService Workerとオフラインキャッシュは動作しません。

## 保護者コード

設定画面で4文字以上の保護者コードを設定してください。

保護者コード設定後も、ミッションにチェックを付ける操作は子どもだけでできます。チェックを外す、クーポンを使用済みにする、ミッションやクーポン候補を編集する操作では承認が必要です。
動物の名前を変更する操作でも承認が必要です。

## サウンド

BGMは最初のユーザー操作後に再生を開始します。ブラウザの自動再生制限に対応するため、ページを開いただけでは鳴りません。

保護者メニュー内の音の設定で、BGM / SE のON/OFFと音量を別々に変更できます。設定は `localStorage` に保存されます。

## 牧場ルール

- 動物は毎日お世話が必要です。
- えさ・水・なでるを1日の中で全部行うと、お世話記録が1日増えます。
- ブラシやおもちゃは、なでる代わりのお世話として使えます。
- 特別えさ、薬、ベッドは体調やなつきを回復させる補助アイテムです。
- どうぶつガチャではレアリティが高い動物ほど出会いにくくなります。
- お世話記録が7日以上で、なつきが高く、体調が良いと、動物が恩返しクーポンを残して旅立ちます。
- 旅立った動物は牧場からいなくなり、アルバムに記録が残ります。
- 世話をしない日が続くと、動物が体調を崩したり、警戒したり、逃げてしまうことがあります。
- プレゼント画面やアルバムは、最初はネタバレにならない表示にしています。

## 画像素材

`images` フォルダの画像を使用します。

- 動物画像: `normal` / `happy` / `hungry` / `sick` / `wary` / `farewell`
- 牧場背景: 通常、春、夏、秋、冬、夜など
- アイテム画像: えさ、水、薬、ブラシ、おもちゃ、ベッドなど
- 表示補助: コイン、ステータス、カレンダースタンプ、クーポン、アルバム枠
- 音源: `assets/audio/bgm` と `assets/audio/se`

## ファイル

- `index.html`: 画面構造
- `styles.css`: 見た目
- `app.js`: ミッション、牧場、動物、コイン、クーポンなどの動き
- `manifest.webmanifest`: PWAインストール情報
- `service-worker.js`: オフラインキャッシュ
- `assets/audio`: BGMと効果音
- `requirements.md`: 要件定義
- `implementation_plan.md`: 実装計画
- `DEPLOY.md`: GitHub Pages公開手順

# 実装計画

## 現在の到達点

静的なブラウザアプリとして、生活習慣チェックと牧場育成を組み合わせた試作版まで実装済み。

### 実装済み

- HTML / CSS / JavaScript の静的アプリ
- 朝、昼、夜のミッション表示
- ミッション達成によるコイン獲得
- チェック取り消し、設定変更、クーポン使用時の保護者コード承認
- localStorage保存
- 牧場背景、柵、動物、アイテム、クーポン、アルバム、カレンダー画像の利用
- 11種類の動物
- どうぶつガチャと動物レアリティ
- スキップ可能なステート制御のどうぶつガチャ演出
- 動物の名前変更と、牧場背景左上の表示中頭数
- 牧場背景下部の動物名ラベル
- 動物ごとの状態画像: normal / happy / hungry / sick / wary / farewell
- コインでガチャ、えさ、水、お世話アイテムを利用
- えさ、水、なでる、特別えさ、薬、ブラシ、おもちゃ、ベッドによる世話
- 日次の状態低下
- 恩返しクーポン
- ネタバレ防止のため、プレゼント画面の初期文言とアルバムタブ表示を制御
- 右上の保護者メニュー
- 牧場の記録アルバム
- PWA manifest、Service Worker、アプリアイコン
- BGM / SE、音量設定、localStorage保存、自動再生制限対応

## 次に拡張しやすい機能

### フェーズ1: 画像版の安定化

- 実ブラウザでの表示確認
- スマホ幅での画像サイズ調整
- 画像読み込み失敗時の代替表示
- 初回プレイ時の導線改善

### フェーズ2: 畑機能

- 畑を購入できるようにする
- 種まき、芽、成長、収穫の段階を追加
- 収穫した作物をえさとして使えるようにする
- 収穫までの日数とミッション継続を連動する

### フェーズ3: 牧場カスタマイズ

- 背景、柵、小屋、花、風車、池、木を購入できるようにする
- 購入した装飾を牧場画面に表示する
- 季節背景やイベント背景を選べるようにする

### フェーズ4: 動物ごとの個性

- 好きなえさ、好きな世話を設定する
- 動物ごとの恩返し条件を少し変える
- 動物ごとの短いメッセージを追加する
- 限定動物やイベント動物を追加する

### フェーズ5: 保護者向け改善

- 週次レポート
- 曜日別ミッション
- ミッションの並び替え
- クーポン利用履歴
- 音源の時間帯切り替え
- インストール案内の改善

## 確認項目

- `node --check app.js` が通る
- `images` フォルダ内の参照画像がすべて存在する
- 11種類の動物に6状態ずつ画像がある
- `manifest.webmanifest` と `service-worker.js` が配信対象に含まれる
- `assets/audio` フォルダが配信対象に含まれる
- ZIPファイルは作成しない
